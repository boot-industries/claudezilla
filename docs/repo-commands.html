Quick reference for common development and deployment commands (v0.5.6)

Click any command to copy to clipboard. Sections are collapsible. Press a to expand all, Esc to collapse all.

Setup & Installation

./install/install-macos.sh Install native messaging host on macOS macOS
./install/install-linux.sh Install native messaging host on Linux Linux
.\install\install-windows.ps1 Install native messaging host on Windows (PowerShell) Windows
.\install\uninstall-windows.ps1 Uninstall native messaging host on Windows Windows
ln -s "$(pwd)/plugin" ~/.claude/plugins/claudezilla-loop Install Claude Code plugin for focus loops Unix

Development

cd host && node index.js Run native messaging host directly (for debugging) All
node mcp/server.js Run MCP server directly (for debugging) All
web-ext run --source-dir extension/ Load extension in Firefox for development All
web-ext run --source-dir extension/ --firefox-profile dev Load extension with named Firefox profile All

Testing & Diagnostics

firefox_diagnose MCP tool: Check socket, auth token, extension status MCP
ls -la $TMPDIR/claudezilla* Check socket and auth token files exist (Unix) Unix
cat $TMPDIR/claudezilla-auth.token View auth token (Unix) Unix
type %LOCALAPPDATA%\claudezilla\auth.token View auth token (Windows) Windows
ls -la ~/.mozilla/native-messaging-hosts/ Check native messaging manifest exists Unix
reg query "HKCU\Software\Mozilla\NativeMessagingHosts\claudezilla" Check Windows registry for native messaging Windows

MCP Server Commands

firefox_create_window Open URL in shared 12-tab pool MCP
firefox_get_tabs List all tabs with ownership info MCP
firefox_screenshot Capture viewport (mutex-protected, 3s timeout) MCP
firefox_get_page_state Fast structured JSON (headings, links, buttons) MCP
firefox_start_loop Begin focus loop for iterative development MCP
firefox_loop_status Check current loop state and iteration MCP

Build & Package

npm pack Create npm package (includes host and MCP) All
web-ext build --source-dir extension/ Build extension XPI for distribution All
web-ext sign --source-dir extension/ --api-key=$AMO_KEY --api-secret=$AMO_SECRET Sign extension for AMO distribution All

Deployment

wrangler pages deploy website --project-name=claudezilla Deploy marketing website to Cloudflare Pages All
cd worker && wrangler deploy Deploy Stripe worker to Cloudflare All
wrangler pages deployment list --project-name=claudezilla List recent Cloudflare Pages deployments All

Git Workflow

git status Show working tree status All
git diff Show unstaged changes All
git log --oneline -10 Show last 10 commits (one line each) All
git checkout -b feature/my-feature Create and switch to new feature branch All
git push -u origin HEAD Push current branch and set upstream All

Troubleshooting

rm -f $TMPDIR/claudezilla.sock Remove stale socket file (Unix) Unix
cat $TMPDIR/claudezilla-debug.log View native host debug log (Unix) Unix
type %LOCALAPPDATA%\claudezilla\debug.log View native host debug log (Windows) Windows
pkill -f "node.*claudezilla" Kill stuck Claudezilla processes (Unix) Unix
taskkill /F /IM node.exe /FI "WINDOWTITLE eq claudezilla*" Kill stuck Claudezilla processes (Windows) Windows
about:debugging#/runtime/this-firefox Firefox debugging page (paste in address bar) All
