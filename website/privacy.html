Privacy Policy

Last updated: January 6, 2025

Overview

Claudezilla is a Firefox extension that enables browser automation for Claude Code CLI. We are committed to protecting your privacy. This policy explains what data the extension accesses and how it is handled.

Data Collection

Claudezilla does not collect, store, or transmit any personal data.

The extension operates entirely locally on your machine. All communication occurs through a Unix socket between the extension and the native messaging host - no data is sent to external servers.

What the Extension Accesses

When you use Claudezilla with Claude Code, the extension may access:

  • Page content: Text, DOM elements, and page structure - only when explicitly requested by Claude
  • Screenshots: Captured and held in memory temporarily, never persisted to disk or transmitted externally
  • Network requests: Monitored for page readiness detection only; request bodies are never captured
  • Console logs: Available on-demand for debugging; opt-in and never transmitted

All accessed data remains on your local machine and is used solely to fulfill automation commands from Claude Code.

Permissions Explained

  • nativeMessaging: Communicate with Claude Code via the native host
  • tabs: Create and manage browser tabs for automation
  • activeTab: Access the current tab when automation is requested
  • <all_urls>: Enable automation on any URL you navigate to
  • webRequest: Detect when pages finish loading
  • storage: Save your preferences (like compression settings)

Third-Party Services

The extension itself uses no third-party services. However:

  • Stripe: If you choose to make a donation via our support page, payment processing is handled by Stripe. See Stripe's Privacy Policy.
  • Helio: Cryptocurrency donations are processed by Helio. See Helio's Privacy Policy.

Analytics

Claudezilla does not include any analytics, telemetry, or tracking. We do not know how many people use the extension or how they use it.

Open Source

Claudezilla is open source under the MIT license. You can review the complete source code at github.com/boot-industries/claudezilla.

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above. Significant changes will be noted in the extension's changelog.

Contact

For privacy-related questions, contact: support@boot.industries
